# Predictive Analytics Using Historical Data

### Project Description
Built a Linear Regression model to predict future values using historical data. This project demonstrates end-to-end Predictive Analytics workflow.

### Dataset Used
Dataset: Daily Total Female Births
Source: Kaggle / GitHub Datasets

### Key Features
- Data Cleaning and Preprocessing
- Linear Regression Model Training
- Model Evaluation using RMSE: 7.81
- Data Visualization with Matplotlib
- 7-Day Future Prediction

### Tools & Libraries Used
Python, Pandas, Numpy, Scikit-learn, Matplotlib
